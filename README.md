# kazalloc
"Kazuya's Malloc", an implementation of C's `malloc` suite. 

## Current Features
### `malloc` 
- Basic functionality
- Splitting existing allocations 

### `free`
- Basic functionality 
- Fusing free blocks

### `print_state`
- Prints all allocated blocks with state & data
